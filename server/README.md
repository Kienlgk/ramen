# Server Skeleton

## Technology Stack
+ NodeJS
+ Express 4.x
+ Typescript 2.x
+ Socket.IO 1.7.x

## How to run
+ `npm install` (`sudo npm install` if linux/osx)
+ edit database config in `src/config.json`
+ `gulp` to test if runable or not
+ `node dist/server` to run server, or
+ `nodemon dist/server` to run & watch
